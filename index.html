About

My name is Luke Rebeiro I am a self-taught Frontend Developer and UI/ UX Designer from the United Kingdom. I enjoy creating and designing websites. I have used various computer design programmes such as Adobe Photoshop, Illustrator, InDesign, and XD. I use these programmes to design my websites before writing them into code. I enjoy writing complex and creative code and always strive to create effective elegant web designs in HTML (HTML5), CSS (CSS3) and Javascript (ES6 and Javascript frameworks such as React).

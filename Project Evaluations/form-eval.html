Brief

I created this enquiry form to show the different user form elements which are used in HTML. The form consists of: user input methods, check-boxes, radio buttons, drop-down menus, comment-box and submission button

This is initial design of the enquiry form which is shown below:

Re-Design

I started by redesigning the logo for the project this is in the image below:

I choose this style of logo because it represents the theme of the form. I used adobe photoshop and illustrator to create the logo, and made it into an SVG file format to be used in future projects. I changed the font and color of the form due to feedback because it was not readable.

I changed background image to black and white to make the font style and colour clearer because the gradient effect was making it harder to read.

As you can see from the images below, it is now clearer for the user to read and navigate through the different areas of the web page.

I added a clickable page when the user has finished the form when they click ‘Are You Ready!’ it will be redirected to a form completion page to indicate the form has worked correctly see image below.

Coding

I used Visual Studio Code as my main coding software. I have used HTML and CSS as my main coding languages.

Conclusion

Following feedback, I have changed the font style, font size, and background image this was to make the text clearer for the user to read. I created and added a logo to make the enquiry from more attractive which engages the user and I created a new web page when the user has submitted the enquiry form. This feedback is important to ensure an enjoyable user experience.
